### 文字化けが発生する

文字コードの指定が必要。
```
<head>
    <meta http-equiv="Content-Type" content="text/html; charset=utf-8"/>
</head>
```
[W3C:HTMLで文字エンコーディングを指定する](https://www.w3.org/International/questions/qa-html-encoding-declarations.ja)
